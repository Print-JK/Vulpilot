# Vulpilot
An AI-assisted penetration testing report generator that transforms raw outputs from common security assessment tools.


## Planned Features

- Parse Nmap XML
- Parse Nuclei JSON
- Normalize findings
- Merge duplicate vulnerabilities
- Generate executive summaries using AI
- Export professional HTML and PDF reports

## Tech Stack

- Python
- FastAPI
- Pydantic
- Jinja2
- WeasyPrint
